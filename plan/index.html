Step by Step

  1. Getting your Hello World locally
    1. Open a new folder and call it "{github_name}.io"
    2. Create a new "index.html" file
    3. Open the folder in vscode
    4. Type Hello I'm {your_name} in the page
    5. Save (cmd + s)
    6. Open a new Google Chrome Tab
    7. Drag the file into the new tab
    8. If the previous step doesn't work, in vscode, right click on the file and click 'copy path', then paste it into chrome
  2. Using different tags
    1. Like word document, we can also use different elements in our page
    2. Like () in Math and "" in English, we can use HTML tags to wrap around our text, giving it special style and properties
    3. You can check out the varioustags on this page
    4. Our "Hello" is a header. To tell the HTML that it is a header, Wrap your "Hello" with the html tags: <h1> Hello I'm {your_name}</h1>
    5. Hit save and reload your chrome page to see your changes
    6. Add a secondary header right below: <h2> Welcome to my page </h2>
    7. Add a third header: <h3> About me </h3>
  3. Getting your "Hello World" online
    1. Let's create a repo (basically a folder) for your code on github
    2. Why? It is a free place to store all our code on cloud; we can access the code on different computers
    3. It also allows us to work on the same codebase with our friends
    4. Lastly, it allows us to host a page for free!
    5. For it to work well, we will have to name the repo {github_name}.io
    6. Now, share the code with github. You can do it either through vscode or by dragging it in manually
    7. This "save" is also called a commit on git
    8. You will have to add a simple message explaining what you are doing. Write something like "my first commit"
    9. Go to the settings > pages >
    10. Wait for a little and you will be able to see your page soon!
    11. While waiting, you can also try to access online vscode from github
    12. Go to the main page of your repo and press "."
  4. List tags (ol, ul, li)
    1. Feel free to add any tags you like into your page
    2. Start a numbered list of items about yourself using <ol></ol>
    3. For each item in the list, we will also need to wrap it in <li></ol> tags
    5. To make life easier, let's update some settings for Vscode: Open settings and search for "Format document on save"
    6. Save and reload the page on chrome (if you are on github, click commit instead, and wait for the changes to show on your page)
    7. "ol" stands for ordered list, and will show a number. You can use "ul", unnumbered list instead
  5. Image tag (img)
    1. We will also add an <img/> tag
    2. Take note that tags with words inside comes in pairs, with an opening tag and a closing tag
    3. Since images doesn't contain any words, we just need a single self-closing tag <img/>
    4. If your tags are not closed properly, the html might break and your page might not load well!
    5. An image tag takes in a source of where to find the image. Let's try adding an image from google.
    6. <img src="{right click the photo and copy the image link}" alt="{describe your photo incase it fails to load}" />
    7. You can also attach a photo locally. This is more reliable
    8. Create a new photo called "assets"
    9. Download the photo into the folder, renaming it "me.jpg" (or "me.png" or other photo formats)
    10. Change the source to "assets/me.jpg"
    11. Check that it still works
  6. Hyperlink tag (a)
    1. The hyperlink tag helps redirect to other pages in your website or other sites
    2. <a href="{link}" taget="_blank">{highlighted_text}</a>
  7. Adding the "head" to our html
    1. The part which we are working on right now is actually the body of the page
    2. A html can also have a head, which helps us pass some other information, such as the title of the page
    3. First, we wrap our whole page in a pair of <body> tags. Everything we have done so far should be between <body> and </body>
  8. Styling
    1. To add our own unique taste to the page, the default styles are not good enough. Let's decorate!
    2. The styles of a .html file is recorded in a .css file
    3. Lets create a new folder called styles, and a new "styles.css" file
